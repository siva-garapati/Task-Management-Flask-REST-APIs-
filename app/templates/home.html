Task Management REST API Documentation

Base URL

http://localhost:5000/

Authentication Endpoints

1. Register User

URL: /auth/register

Method: POST

Request Body:

{
  "username": "your_username",
  "password": "your_password"
}

Success Response:

{
  "msg": "Registration successful"
}

2. Login User

URL: /auth/login

Method: POST

Request Body:

{
  "username": "your_username",
  "password": "your_password"
}

Success Response:

{
  "msg": "Login successful",
  "token": "JWT_ACCESS_TOKEN"
}

Task Endpoints (JWT protected)

1. Create Task

URL: /api/addtask

Method: POST

Request Body:

{
  "title": "Finish Flask REST API",
  "content": "Implement CRUD endpoints",
  "status": "Pending",
  "deadline": "2025-09-20"
}

2. Get All Tasks

URL: /api/tasks

Method: GET

3. Get Single Task

URL: /api/tasks/<task_id>

Method: GET

4. Update Task

URL: /api/tasks/<task_id>

Method: PUT

5. Delete Task

URL: /api/tasks/<task_id>

Method: DELETE

Notes

  • JWT Token: Include in headers for all task routes: Authorization: Bearer <token>
  • Deadline format: YYYY-MM-DD
  • Status values: Pending (default) or Completed
  • IDs (task_id, user_id) are UUID strings
